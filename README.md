# 📝 To-Do List CLI Application

A simple **Command-Line To-Do List App** built using Python.
This beginner-friendly project allows users to manage daily tasks directly from the terminal.

---

## 🚀 Features

* ✅ Add Tasks
* 📋 View Tasks
* ❌ Delete Tasks
* 🖥 Menu-Driven Interface
* ⚠ Error Handling for Invalid Inputs

---

## 🛠 Technologies Used

* Python 3
